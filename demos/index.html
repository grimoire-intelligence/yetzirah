Yetzirah

The world of formation.

AI-native, unstyled Web Components with Material behavior. Zero dependencies, framework-agnostic, accessibility-first.

Component Demos

Each component has its own demo page showing usage examples and variations. All demos work by opening the HTML file directly in a browser - no build step required.

  • Accordion - Coordinated disclosures with exclusive mode
  • Button - Polymorphic button/anchor component
  • Dialog - Modal dialog with focus management
  • Disclosure - Expandable content with aria-expanded
  • Tabs - Accessible tabbed interface with keyboard navigation
  • Tooltip - Positioned hint text with delay and auto-flip
  • [Coming Soon] Menu - Positioned menu with keyboard navigation
  • [Coming Soon] Autocomplete - Text input with filtering and selection

Philosophy

  • Platform-native. Web Components extend the DOM rather than replacing it.
  • AI-native. The entire framework fits in a single LLM context window.
  • Framework-agnostic. Works with React, Vue, Svelte, Angular, or vanilla HTML.
  • No-pinionated. Provides behavior and accessibility. You provide appearance.
  • Readable source. Every component fits in a single file. Implementation is documentation.

Quick Start

# Install via npm
npm install @yetzirah/core

# With React wrappers
npm install @yetzirah/core @yetzirah/react

Vanilla HTML

<script type="module" src="https://cdn.yetzirah.dev/core.js"></script>

<ytz-button href="/dashboard" class="ph3 pv2 br2 white bg-blue">
  Go to Dashboard
</ytz-button>

React

import { Button } from '@yetzirah/react'

function App() {
  return (
    <Button href="/dashboard" className="ph3 pv2 br2 white bg-blue">
      Go to Dashboard
    </Button>
  )
}
