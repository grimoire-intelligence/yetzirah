IconButton

Icon-only button requiring aria-label for accessibility. Includes optional built-in tooltip support.

Basic Usage

Always provide an aria-label for screen readers. 

<ytz-icon-button aria-label="Search" onclick="handleSearch()">
  <svg>...</svg>
</ytz-icon-button>

With Tooltip

Add the tooltip attribute to show a tooltip on hover/focus using the aria-label text. 

<ytz-icon-button aria-label="Edit" tooltip>
  <svg>...</svg>
</ytz-icon-button>

Disabled State

Disabled icon buttons are non-interactive. 

<ytz-icon-button aria-label="Save" disabled>
  <svg>...</svg>
</ytz-icon-button>

Sizes

Control size via CSS classes on the wrapper.

Note: Sizing with custom CSS is straightforward (target .ytz-icon-button-inner and svg). Tachyons is less convenient here since you'd need classes on both the element and the SVG, and Tachyons' sizing scale lacks common icon button sizes like 40px.

Size CSS Class
Small (32px)icon-button-small
Medium (40px)(default)
Large (48px)icon-button-large

Colors

Apply color classes to customize the icon color. 

.icon-button-primary .ytz-icon-button-inner { color: #357edd; }
.icon-button-success .ytz-icon-button-inner { color: #19a974; }
.icon-button-error .ytz-icon-button-inner { color: #ff4136; }

Accessibility

Important: Always provide aria-label since there's no visible text. A console warning appears if the label is missing.

  • aria-label on inner button for screen readers
  • Tooltip uses role="tooltip"
  • Focus-visible styles for keyboard navigation
  • Console warning if aria-label is missing

MUI Migration

MUI Yetzirah
<IconButton><Icon /></IconButton> <ytz-icon-button aria-label="..."><svg>...</svg></ytz-icon-button>
<IconButton disabled> <ytz-icon-button disabled>
<Tooltip title="..."><IconButton> <ytz-icon-button tooltip>
size="small" CSS (see Sizes section)

React Usage

import { IconButton } from '@yetzirah/react'
import { SearchIcon, SettingsIcon } from './icons'

function Toolbar() {
  return (
    <div>
      <IconButton aria-label="Search" onClick={handleSearch}>
        <SearchIcon />
      </IconButton>

      <IconButton aria-label="Settings" tooltip>
        <SettingsIcon />
      </IconButton>

      <IconButton aria-label="Disabled" disabled>
        <SettingsIcon />
      </IconButton>
    </div>
  )
}
