Tabs

Accessible tabbed interface with keyboard navigation.

Overview Features Usage

Yetzirah is a headless component library built with Web Components.
  • Zero dependencies
  • Framework agnostic
  • Accessible by default
  • Minimal bundle size

Load from CDN, use with any framework, or vanilla JS!

 
<ytz-tabs value="tab1">
  <ytz-tab panel="tab1">Tab 1</ytz-tab>
  <ytz-tabpanel id="tab1">Content</ytz-tabpanel>
</ytz-tabs>

Accordion

Coordinated disclosure components with optional exclusive mode.

Yetzirah is a headless component library that provides accessible, unstyled Web Components for common UI patterns.
Simply add a script tag pointing to the CDN bundle, and all custom elements are automatically registered. No build step needed!
Components are unstyled by default. Use Tachyons, your own CSS, or any CSS framework to style them. 
<ytz-accordion exclusive>
  <ytz-accordion-item>
    <button>Question</button>
    <div>Answer</div>
  </ytz-accordion-item>
</ytz-accordion>

Usage

Script Tag (Simplest)

<!-- Load all components -->
<script type="module" src="https://cdn.jsdelivr.net/npm/@grimoire/yetzirah-core@latest/cdn/core.js"></script>

<!-- Or just Tier 1 (~6KB gzipped) -->
<script type="module" src="https://cdn.jsdelivr.net/npm/@grimoire/yetzirah-core@latest/cdn/tier1.js"></script>

Import Maps (npm-like DX)

<script type="importmap">
{
  "imports": {
    "yetzirah": "https://cdn.jsdelivr.net/npm/@grimoire/yetzirah-core@latest/cdn/core.js"
  }
}
</script>

<script type="module">
  import 'yetzirah';
  // All components now available!
</script>

Individual Components

<!-- Load only what you need -->
<script type="module" src="https://cdn.jsdelivr.net/npm/@grimoire/yetzirah-core@latest/cdn/dialog.js"></script>
<script type="module" src="https://cdn.jsdelivr.net/npm/@grimoire/yetzirah-core@latest/cdn/tabs.js"></script>
