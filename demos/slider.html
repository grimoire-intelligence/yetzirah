Slider

Accessible slider/range component with keyboard control, touch support, and customizable min/max/step values.

Basic Usage

A simple slider with default range 0-100.

50 
<ytz-slider min="0" max="100" value="50"></ytz-slider>

With Step

Use the step attribute to control increment size.

50
50 
<ytz-slider min="0" max="100" value="50" step="10"></ytz-slider>

Custom Range

Set custom min/max values for different use cases.

20°C
5 
<!-- Temperature range -20 to 50 -->
<ytz-slider min="-20" max="50" value="20"></ytz-slider>

<!-- Volume goes to 11 -->
<ytz-slider min="0" max="11" value="5"></ytz-slider>

Disabled State

Disabled sliders cannot be interacted with.

75 (disabled) 
<ytz-slider min="0" max="100" value="75" disabled></ytz-slider>

Sizes

Control size via CSS classes.

Colors

Apply color classes for different contexts. 

.slider-success .ytz-slider-track {
  background: linear-gradient(to right,
    #19a974 var(--slider-percent),
    #e0e0e0 var(--slider-percent)
  );
}
.slider-success .ytz-slider-thumb {
  background: #19a974;
}

Dynamic Color

Use CSS to grade the slider color based on value. The --slider-percent custom property updates live as the slider moves.

25% 
/* Track shows full gradient */
.slider-dynamic .ytz-slider-track {
  background: linear-gradient(to right,
    #19a974 0%,    /* green */
    #ffcc00 40%,   /* yellow */
    #ff6300 70%,   /* orange */
    #ff4136 100%   /* red */
  );
}

/* Thumb color interpolates based on position */
.slider-dynamic .ytz-slider-thumb {
  background: color-mix(
    in oklch,
    #19a974 calc(100% - var(--slider-percent)),
    #ff4136 var(--slider-percent)
  );
}

Full keyboard support for accessibility.

Key Action
Arrow Right/UpIncrease by step
Arrow Left/DownDecrease by step
Page UpIncrease by 10× step
Page DownDecrease by 10× step
HomeGo to minimum
EndGo to maximum

Programmatic Control

Control slider value via JavaScript.

50 
// Set value
slider.value = 75

// Get value
console.log(slider.value)

// Get/set other properties
slider.min = 0
slider.max = 200
slider.step = 5
slider.disabled = true

Accessibility

The slider automatically manages ARIA attributes:

  • role="slider" - Identifies as a slider
  • aria-valuemin - Minimum value
  • aria-valuemax - Maximum value
  • aria-valuenow - Current value
  • aria-disabled - Disabled state
  • tabindex="0" - Makes it focusable

React Usage

import { Slider } from '@grimoire/yetzirah-react'

function VolumeControl() {
  const [volume, setVolume] = useState(50)

  return (
    <div>
      <Slider
        min={0}
        max={100}
        value={volume}
        onChange={(e) => setVolume(e.detail.value)}
      />
      <span>{volume}%</span>
    </div>
  )
}
