Select

Dropdown select with trigger button and listbox. Classic dropdown pattern.

Basic Usage

Click the trigger to open, select an option to close.

Apple Banana Cherry Date 
<ytz-select placeholder="Select a fruit...">
  <ytz-option value="apple">Apple</ytz-option>
  <ytz-option value="banana">Banana</ytz-option>
  <ytz-option value="cherry">Cherry</ytz-option>
</ytz-select>

Multi-Select

Add multiple to allow multiple selections.

Red Green Blue Yellow 
<ytz-select multiple placeholder="Select colors...">
  <ytz-option value="red">Red</ytz-option>
  <ytz-option value="green">Green</ytz-option>
  <ytz-option value="blue">Blue</ytz-option>
</ytz-select>

Programmatic Control

Apple Banana Cherry 
const select = document.querySelector('ytz-select')

select.show()           // Open dropdown
select.hide()           // Close dropdown
select.toggle()         // Toggle open/closed
select.clear()          // Clear selection
select.value = 'apple'  // Set value programmatically

Keyboard Support

Key Action
Enter / Space / Arrow DownOpen dropdown (on trigger)
Arrow Down / Arrow UpNavigate options
Enter / SpaceSelect option
EscapeClose dropdown
TabClose and return to trigger

API Reference

Attributes

AttributeTypeDescription

multipleBooleanAllow multiple selections

placeholderStringTrigger placeholder text

Events

EventDetailDescription

React Usage

import { useState } from 'react'
import { Select, Option } from '@grimoire/yetzirah-react'

function StatusPicker() {
  const [status, setStatus] = useState('')

  return (
    <Select value={status} onChange={setStatus} placeholder="Select status...">
      <Option value="draft">Draft</Option>
      <Option value="review">In Review</Option>
      <Option value="published">Published</Option>
    </Select>
  )
}

// Multi-select
function TagPicker() {
  const [tags, setTags] = useState([])

  return (
    <Select multiple value={tags} onChange={setTags}>
      <Option value="urgent">Urgent</Option>
      <Option value="bug">Bug</Option>
      <Option value="feature">Feature</Option>
    </Select>
  )
}
