Disclosure

Expandable content with aria-expanded. Animation-friendly open/close states.

Basic Usage

Click the button to toggle content visibility. The disclosure handles ARIA attributes automatically.

This content is hidden by default. Click the button to toggle visibility. The component automatically manages aria-expanded and aria-controls attributes. 

<ytz-disclosure>
  <button class="ph3 pv2 br2 bg-blue white bn pointer">
    Show Details
  </button>
  <div>
    <div class="pa3 bg-light-gray br2">
      Hidden content...
    </div>
  </div>
</ytz-disclosure>

Initially Open

Use the open attribute to start with content visible.

This content is visible by default because the open attribute is present. 

<ytz-disclosure open>
  <button>Hide Details</button>
  <div>Visible content...</div>
</ytz-disclosure>

Animated Expand/Collapse

The optional stylesheet provides smooth height and opacity transitions. Add the ytz-disclosure-icon class to rotate chevrons, or ytz-disclosure-indicator for +/- toggles.

This content animates smoothly when toggled. The optional stylesheet uses CSS grid animation for natural height transitions without JavaScript. 

<ytz-disclosure>
  <button class="... flex items-center">
    <span>Toggle</span>
    <span class="ytz-disclosure-icon ml2">&#9660;</span>
  </button>
  <div>
    <div class="pa3">Content...</div>
  </div>
</ytz-disclosure>

Optional Stylesheet

Yetzirah takes no styling opinions—but animated expand/collapse is basic interaction behavior that most applications will want. We ship an optional stylesheet that provides this behavior while staying true to the "unstyled" philosophy. 

// Import the optional stylesheet
import '@grimoire/yetzirah-core/disclosure.css'

// Or in HTML
<link rel="stylesheet" href="@grimoire/yetzirah-core/disclosure.css">
View stylesheet source 
/* Content panel - collapsed state */
ytz-disclosure > :last-child {
  display: grid;
  grid-template-rows: 0fr;
  opacity: 0;
  transition: grid-template-rows 0.25s ease-out, opacity 0.2s ease-out;
}

/* Override hidden attribute to enable animation */
ytz-disclosure > :last-child[hidden] {
  display: grid !important;
}

/* Inner wrapper needed for grid animation */
ytz-disclosure > :last-child > * {
  overflow: hidden;
}

/* Content panel - expanded state */
ytz-disclosure[open] > :last-child {
  grid-template-rows: 1fr;
  opacity: 1;
}

/* Chevron indicator rotation */
ytz-disclosure .ytz-disclosure-icon {
  display: inline-block;
  transition: transform 0.2s ease;
}

ytz-disclosure[open] .ytz-disclosure-icon {
  transform: rotate(180deg);
}

/* Plus/minus indicator toggle */
ytz-disclosure .ytz-disclosure-indicator::after {
  content: '+';
}

ytz-disclosure[open] .ytz-disclosure-indicator::after {
  content: '\2212'; /* minus sign */
}

How It Works

The stylesheet uses CSS grid's grid-template-rows trick for animating height. This requires one extra wrapper div inside the content panel: 

<ytz-disclosure>
  <button>Toggle</button>
  <div>                    <!-- Grid container (animated) -->
    <div class="pa3">      <!-- Inner wrapper (overflow:hidden) -->
      Actual content...
    </div>
  </div>
</ytz-disclosure>

FAQ Pattern

Multiple disclosures work independently. For coordinated behavior (only one open at a time), use the Accordion component instead.

Yetzirah is an AI-native, unstyled Web Component library providing Material-like behavior without the styling overhead. It's designed to fit entirely within an LLM context window.

Yetzirah ships no styles, so style encapsulation solves nothing. Light DOM allows Tachyons and user CSS to work naturally without the complexity of ::part() selectors.

Target is <10kb gzipped for all Tier 1 components. Each component is under 200 lines of code.

Disclosures can contain any HTML content including forms, images, and other components.

API Reference

Attributes

Attribute Type Description
open Boolean When present, content is visible

Events

Event Detail Description
toggle { open: boolean } Fired when open state changes

CSS Classes (Optional Stylesheet)

Class Description
ytz-disclosure-icon Rotates 180deg when open (for chevrons)
ytz-disclosure-indicator Shows + when closed, - when open

JavaScript API

const disclosure = document.querySelector('ytz-disclosure')

// Read state
console.log(disclosure.open) // false

// Set state
disclosure.open = true

// Toggle programmatically
disclosure.toggle()

// Listen for changes
disclosure.addEventListener('toggle', (e) => {
  console.log('Now open:', e.detail.open)
})

React Usage

import { useState } from 'react'
import { Disclosure } from '@grimoire/yetzirah-react'
import '@grimoire/yetzirah-core/disclosure.css' // Optional animations

function App() {
  const [open, setOpen] = useState(false)

  return (
    <Disclosure open={open} onToggle={setOpen}>
      <button className="ph3 pv2 br2 bg-blue white bn pointer">
        {open ? 'Hide' : 'Show'} Details
      </button>
      <div>
        <div className="pa3 bg-light-gray mt2 br2">
          Content here...
        </div>
      </div>
    </Disclosure>
  )
}

// Uncontrolled with defaultOpen
<Disclosure defaultOpen onToggle={(open) => console.log(open)}>
  ...
</Disclosure>

Migrating from MUI Collapse

MUI

<Button onClick={() => setOpen(!open)}>
  Toggle
</Button>
<Collapse in={open}>
  <div>Content</div>
</Collapse>

Yetzirah

<Disclosure open={open} onToggle={setOpen}>
  <button>Toggle</button>
  <div><div>Content</div></div>
</Disclosure>

Key difference: Yetzirah's Disclosure includes both the trigger and content, managing their ARIA relationship automatically.
