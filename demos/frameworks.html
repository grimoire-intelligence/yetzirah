Framework Comparison

Yetzirah provides first-class wrappers for Vue, Svelte, and Angular. Each wrapper provides idiomatic bindings that feel native to the framework.

Toggle: Binding Syntax Comparison

● Vue 3

<Toggle v-model:checked="enabled">
  Dark mode
</Toggle>

● Svelte

<Toggle bind:checked={enabled}>
  Dark mode
</Toggle>

● Angular

<ytz-toggle [(ngModel)]="enabled">
  Dark mode
</ytz-toggle>

● React

<Toggle
  checked={enabled}
  onChange={e => setEnabled(e.detail.checked)}
>
  Dark mode
</Toggle>

Slider: Two-way Binding

● Vue 3

<Slider
  v-model="volume"
  :min="0"
  :max="100"
/>

● Svelte

<Slider
  bind:value={volume}
  min={0}
  max={100}
/>

● Angular

<ytz-slider
  [(ngModel)]="volume"
  [min]="0"
  [max]="100"
></ytz-slider>

● React

<Slider
  value={volume}
  min={0}
  max={100}
  onChange={e => setVolume(e.detail.value)}
/>

DataGrid: Complex Component

● Vue 3

<DataGrid
  :data="users"
  :columns="[
    { key: 'name', header: 'Name', sortable: true },
    { key: 'email', header: 'Email' }
  ]"
  @sort="handleSort"
  @rowclick="handleRowClick"
/>

● Svelte

<DataGrid
  {data}
  {columns}
  on:sort={handleSort}
  on:rowselect={handleRowClick}
/>

● Angular

<ytz-datagrid
  [data]="users"
  [columns]="columns"
  (sort)="handleSort($event)"
  (rowClick)="handleRowClick($event)"
></ytz-datagrid>

● React

<DataGrid
  data={users}
  columns={columns}
  onSort={handleSort}
  onRowClick={handleRowClick}
/>
